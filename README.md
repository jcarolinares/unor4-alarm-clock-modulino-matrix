# Uno R4 Alarm Clock Modulino Matrix

An alarm clock created with Arduino UNO R4 and four Modulino Matrix with multple features.

## Hardware

* 1x Arduino UNO R4 Wifi
* 4x Modulino Matrix
* 1x Modulino Knob
* 1x Modulino Buzzer
* Multiple Qwiic connector cables (normally included with the Modulinos)

## Software
* Arduino IDE, cli or another compatible Arduino Program.
